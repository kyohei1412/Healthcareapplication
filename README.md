# Healthcare Application（健康管理アプリ）

## 概要
Android Studio（Kotlin）で作成した健康管理アプリです。  
血圧などの健康データを記録し、一覧表示・検索できるようにしました。

## 主な機能
- 健康データの登録（例：最高血圧 / 最低血圧 など）
- 登録データの一覧表示
- データの並び替え（新しい順）

## 使用技術
- Kotlin
- Android Studio
- Realm Kotlin（ローカルデータベース）
- RecyclerView（一覧表示）
- ViewBinding

## 工夫した点
- 入力しやすいUIになるように項目を整理しました
- Realmを使って端末内にデータを保存し、アプリを閉じても記録が残るようにしました

- ## 動作環境
- Android 10 以上（例）
